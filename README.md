# TimeBase

Calendar analytics in the browser. Upload iCal data, explore activity stats, filters, and charts in a Next.js app.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Environment

Optional: set `GEMINI_API_KEY` in `.env.local` for the AI chat feature.

## Stack

Next.js (App Router), React, Tailwind CSS, and client-side calendar parsing for privacy-friendly analysis.
